# DTF Radar

A lightweight competitor intelligence tool for the DTF market.

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- App Router

## Goal
Track competitor pricing, product launches, promotions, and messaging changes across DTF-related websites.

## Included
- Product landing page scaffold
- MVP PRD in `docs/`

## Local dev

```bash
npm install
npm run dev
```

Open http://localhost:3000
