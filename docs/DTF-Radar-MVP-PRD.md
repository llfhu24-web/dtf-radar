# DTF Radar MVP PRD

> Moved from workspace planning doc for in-project reference.

See the full product requirements below.

## 1. 产品名称
**DTF Radar**  
副标题：**Track competitor pricing, product launches, and promotions in the DTF market.**

## 2. 产品定位
### 2.1 一句话定位
为 DTF/打印膜/热转印耗材行业卖家与品牌，提供 **竞品网站变化监控 + AI 情报摘要** 服务。

### 2.2 核心价值
帮助用户自动发现竞争对手的：
- 新品上线
- 价格变化
- 促销变化
- 页面卖点变化
- 重点分类策略变化

并把这些变化转化为 **可执行情报**，替代人工盯盘。

## 3. 目标用户
### 3.1 核心用户画像
#### 用户 A：DTF 耗材卖家负责人
- 有自己的网站或分销渠道
- 关心同行价格、上新和卖点变化
- 没时间每天手工看很多网站

#### 用户 B：独立站运营 / 增长负责人
- 负责海外 DTF 相关独立站
- 关注 landing page、产品页、促销活动、SEO 变化
- 希望更快响应市场变化

#### 用户 C：渠道商 / 批发商
- 关注品牌新品、MOQ、批发政策、价格带变化
- 需要行业情报辅助采购与报价

## 4. 用户痛点
1. 竞争对手改价 / 促销，无法第一时间知道  
2. 竞争对手上线新品，发现滞后  
3. 无法系统跟踪同行卖点变化  
4. 人工监控效率低，信息碎片化  
5. 看到页面变化，也不知道“是否重要”

## 5. MVP 目标
- 添加竞品网站
- 自动发现重要页面
- 定期抓取页面快照
- 检测关键变化
- 生成 AI 摘要
- 通过 Dashboard + 邮件 / Telegram 推送结果

## 6. MVP 核心功能
- 竞品网站管理
- 页面发现
- 页面抓取与快照存档
- 变化检测
- AI 变化摘要
- Dashboard
- 每日简报

## 7. 核心场景
### 发现同行降价
- 60cm hot peel film 从 $89 降到 $79
- 降幅 11.2%
- 页面 banner 新增 “Free shipping over $99”

### 发现同行上新
- 新增 “Sample Pack” 产品页
- 强调 beginner / testing / starter

### 发现卖点切换
- 多个产品页增加 “anti-static” / “hot peel in 3 sec”
